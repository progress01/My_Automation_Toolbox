# 📄 Office 批次轉 PDF 工具 (Office to PDF Batch Converter)

本工具為輕量級的本地端自動化腳本，透過 Windows COM 介面在背景呼叫 Microsoft Office，將指定資料夾內的 Word 文件與 PowerPoint 簡報批次轉換為 PDF 格式。適合用於快速處理大量文書檔案，省去手動另存新檔的繁瑣步驟。

## 1. 需準備的輸入檔案格式

本程式會自動掃描指定資料夾，並僅針對以下微軟 Office 檔案格式進行處理：

* **Word 文件：** `.doc`, `.docx`
* **PowerPoint 簡報：** `.ppt`, `.pptx`

**資料夾配置範例：**
請將需要轉換的檔案集中放置於同一個資料夾中（例如程式碼中預設的 `C:\Users\USER\Desktop\上班資料夾\轉PDF丟這邊`）。

* ✅ `2024年度專案報告.docx` (將被處理)
* ✅ `Q3_業務簡報.pptx` (將被處理)
* ❌ `財務報表.xlsx` (Excel 檔將被自動略過)
* ❌ `已經轉好的文件.pdf` (現有的 PDF 檔將被自動略過)

## 2. 預期的產出結果

執行程式後，將會在**同一個資料夾內**生成對應的 PDF 檔案。

* **無損輸出**：原始的 Word 或 PPT 檔案將會被完整保留，不會被覆蓋或刪除。
* **檔名一致**：新生成的 PDF 會繼承原始檔案的名稱。
* *範例：* `2024年度專案報告.docx` ➡️ `2024年度專案報告.pdf`


* **終端機日誌**：執行過程中，命令提示字元 (Terminal) 會即時顯示轉換進度與成功/失敗狀態，並在結束時確保 Office 背景程序被安全關閉。

## 3. 注意事項

在使用本工具前，請務必確認以下運行環境與設定限制：

* **作業系統與軟體依賴**：
* 本腳本依賴 Windows API，**僅限 Windows 作業系統**執行（不支援 macOS 或 Linux）。
* 電腦**必須安裝正版 Microsoft Word 與 PowerPoint**，否則程式無法呼叫應用程式進行轉換。


* **必備第三方套件**：
* 執行前需確保已安裝 `pywin32` 套件。若尚未安裝，請在終端機輸入以下指令進行安裝：
```bash
pip install pywin32

```




* **路徑設定規則**：
* 請務必開啟腳本，將 `TARGET_FOLDER` 變數修改為您實際要轉換的資料夾路徑。
* 為避免 Windows 路徑中的反斜線 `\` 造成字元跳脫錯誤（如 `\n`, `\t`），路徑字串前方請務必保留 `r` 前綴（例如：`r"C:\您的\路徑"`）。


* **執行時建議**：
* 執行程式前，**建議先關閉欲轉換的 Word 或 PPT 檔案**，以免因檔案被系統鎖定而導致轉換失敗。
* 程式執行期間會在背景呼叫 Office，若檔案數量龐大，請耐心等候終端機顯示「所有轉換作業結束！」。

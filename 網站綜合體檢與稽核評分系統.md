# 📄 網站綜合體檢與稽核評分系統

本系統為專為大規模網站巡檢打造的自動化體檢工具。透過靜態解析與 `DrissionPage` 動態渲染雙引擎，能深度稽核網站的無障礙規範 (A11y)、檔案開放格式 (ODF)、連結有效性、系統效能與前端資安風險。系統具備高度容錯與人機協作機制，並能一鍵產出供機關查閱的 Excel 追蹤表、HTML 互動儀表板與 Word 評分總表。

## 1. 需準備的輸入資料 (Input Data)

本系統提供四種不同的作業模式，對應不同的輸入需求。**請特別注意 Word 公版的準備**：

### A. 必備底層檔案

* 📄 **`template.docx` (Word 評分公版)**
* **位置**：必須放置於與 Python 程式碼**同一個資料夾目錄下**。
* **說明**：系統在產出最終的 Word 打分報表時，會直接讀取此空白公版進行資料填充。若遺失此檔案，系統將無法產出 Word 報表（但不會導致程式崩潰）。

### B. 各分頁輸入格式

| 執行模式 | 需準備的檔案與格式要求 |
| --- | --- |
| **1. 單站/SPA 動態解析** | **無須檔案**。直接於介面輸入完整的網址（如 `[https://www.example.gov.tw/](https://www.example.gov.tw/)`）。 |
| **2. 清單匯入 (.txt)** | **純文字檔 (`.txt`)**。<br>

<br>・可使用 `#` 作為單行註解 (系統會自動略過)。<br>

<br>・支援 Tab 鍵分隔，系統會自動抓取該行的**最後一個欄位**作為網址。<br>

<br>*(範例：`環境部首頁 [Tab鍵] [https://www.moenv.gov.tw/](https://www.moenv.gov.tw/)`)* |
| **2. 多站批次排程 (.xlsx)** | **Excel 或 CSV 檔**。<br>

<br>・**A 欄 (第一欄)**：網站名稱 (用於報表檔名與標題)。<br>

<br>・**B 欄 (第二欄)**：網站完整網址。<br>

<br>・**C 欄之後**：可自由作為內部備註，系統會自動忽略。<br>

<br>*(註：系統具備防呆機制，無須刻意刪除表頭)* |
| **3. 單頁錯誤檢查** | 載入本系統自己產出的 `網站綜合體檢追蹤表_*.xlsx`，系統將自動解析裡面的錯誤座標。 |

## 3. 預期的產出結果 (Expected Output)

每次執行任務，系統都會在您指定的儲存位置（預設為「我的文件」）建立一個專屬資料夾，命名格式為：`[稽核/SPA]_[網站名稱]_[時間戳記]`。
資料夾內將包含四份極具價值的分析報表：

1. **📊 網站綜合體檢追蹤表 (`.xlsx`)**
* 包含四個分頁：「檔案與連結異常」、「無障礙問題確認」、「系統與連線追蹤」、「共通性無障礙問題確認」。
* 所有失效連結、違規檔案格式 (非 ODF) 皆已依據嚴重程度進行儲存格顏色標記，並附有具體的修復建議與 DOM 座標。


2. **📝 網站檢核評分表 (`.docx`)**
* 完美套用您提供的 `template.docx` 格式，自動填入各項次的「得分 / 扣分」，並將錯誤明細（如：缺乏無障礙標章、語系錯亂、死連結）寫入對應的內容說明欄位。


3. **🗺️ 網站三層結構地圖 (`.xlsx`)**
* 自動盤點網站的 L1、L2、L3 架構，並附帶每一頁的「檔案數量」、「圖片數量」、「內文字數」與「健康度指標」。


4. **📈 網站檢核評分表 (`.html`)**
* 單檔互動式網頁儀表板，具備關鍵字搜尋與錯誤類型下拉篩選功能，方便稽核人員快速查閱。



*(註：若在批次模式下遇到被隔離的 SPA 網站，根目錄將額外產生一份 `spa_pending_list.txt` 供後續手動補測。)*

## 4. 執行環境與注意事項 (Caveats & Requirements)

### 環境依賴 (Dependencies)

本系統引入了多項強大的底層套件以提升解析與存檔效能。請開啟終端機執行以下安裝指令：

```bash
pip install pandas beautifulsoup4 DrissionPage openpyxl fake-useragent lxml python-docx

```

*(註：`python-docx` 是產出 Word 報表的關鍵；`lxml` 則能大幅提升大型 HTML 的解析速度。)*

### ⚠️ 執行與系統注意事項

* **SQLite 離線快取機制 (`temp_cache.db`)**：
* 系統在運行時會於同目錄下頻繁讀寫 `temp_cache.db` 系列檔案。這是先進的記憶體/硬碟混合快取技術，確保同一個網址（如首頁）被掃描數十次時，只有第一次會消耗網路流量。**程式結束時會自動銷毀這些暫存檔，若遇當機殘留可直接手動刪除，毫無安全疑慮**。


* **免安裝版 Chrome 支援**：
* 只要將免安裝版 Chrome 放置於同目錄的 `ChromePortable/chrome.exe`，系統便會強制調用該瀏覽器，**完全不干擾您本機主系統的工作環境**，非常適合裝在隨身碟中隨插即用。


* **人機協作與防護攔截 (SOS 救援)**：
* 當爬蟲遇到 Cloudflare (WAF)、圖形驗證碼或會員登入牆時，系統**不會**直接崩潰報錯，而是會暫停運作，並在畫面頂部彈出「紅色警告橫幅」。您可以從容地手動輸入驗證碼或帳密，完成後點擊該紅色區塊，爬蟲就會無縫接管並繼續深入掃描。


* **Excel 鎖定防護**：
* 若在產出報表時，您剛好打開了舊的 Excel 檔案導致檔案被系統鎖定，程式不會崩潰，而是會自動在檔名後方加上隨機的 4 位數字（如 `_4512.xlsx`）以保護您耗時爬取下來的珍貴數據。

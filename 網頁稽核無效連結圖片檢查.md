# 📄 網頁稽核三階段整合系統 (Three-Stage Web Link Auditor)

本系統為一款專為大規模網頁資料稽核設計的自動化工具。結合了**背景無頭爬蟲（自動截圖與標註）**、**單頁面視覺化 HTML 報表**，以及**半自動遙控除錯引擎**。程式具備跨平台支援與 SQLite 斷點續傳機制，能大幅減輕專案管理與工程端在網站重構、網址遷移或死檔檢測時的人工作業負擔。

## 1. 需準備的輸入檔案格式

執行程式時，系統會彈出視窗要求選擇一份資料檔（支援 **`.xlsx`** 或 **`.csv`** 格式）。
檔案內**必須包含標題列**，且具備以下三個核心欄位（系統會依據這些欄位名稱進行抓取）：

| 編號 | 首次發現來源 (父頁面) | 連結網址 (子頁面/目標網址) |
| --- | --- | --- |
| A001 | `[https://example.gov.tw/news](https://example.gov.tw/news)` | `[https://example.gov.tw/news/detail/123](https://example.gov.tw/news/detail/123)` |
| A002 | `[https://test-site.com/about](https://test-site.com/about)` | `[https://test-site.com/contact-us](https://test-site.com/contact-us)` |
| A003 | `[https://demo.com/home](https://demo.com/home)` | `[https://external-link.com/file.pdf](https://external-link.com/file.pdf)` |

* **編號**：用作資料庫追蹤與截圖命名的唯一識別碼。
* **首次發現來源**：爬蟲會先進入此網址，尋找目標連結並進行「紅框高亮發光」標註與截圖。
* **連結網址**：爬蟲預計要點擊或檢測的最終目標網址，用於驗證是否失效（404、403 等）。

## 2. 預期的產出結果

載入檔案後，程式會在同層目錄下自動建立一個名為 `AuditResult_{您的檔名}` 的專屬專案資料夾，所有產出皆會集中於此：

1. **`audit_progress.db` (核心資料庫)**：
紀錄每一筆任務的執行狀態、錯誤原因（如：404、載入逾時）與截圖路徑。支援中斷續傳，關閉程式後下次執行會自動跳過已完成的項目。
2. **`evidence/` (佐證截圖資料夾)**：
存放巡檢過程中的所有截圖。包含父頁面標註截圖 (`_Source.png`) 與子頁面結果截圖 (`_Child.png` 或 `_Fail.png`)。
3. **`QA_Report_{檔名}.html` (視覺化儀表板)**：
透過 [模式 2] 生成。這是一份互動式的 QA 網頁，將您的 Excel 資料與資料庫抓取的截圖並列顯示，並自動標示成功（綠）、隱藏（黃）、失敗（紅）等狀態。支援勾選異常項目並一鍵複製編號。
4. **`Final_Report_{檔名}_{時間}.xlsx` (最終彙整報表)**：
透過 [模式 4] 生成。將原始匯入的資料表與系統稽核的結果（狀態、備註、截圖相對路徑）進行合併，輸出為立即可交件的 Excel 結案報告。

## 3. 注意事項

* **環境與相依套件**：
執行前請確保已安裝相關套件。可透過終端機執行以下指令：
```bash
pip install pandas openpyxl selenium webdriver-manager

```


*註：`openpyxl` 是處理 Excel `.xlsx` 檔案讀寫的必要套件。*
* **跨平台互動差異 (遙控引擎)**：
在執行 [模式 3：精準打擊] 時，若於 Windows 系統執行，程式會啟用無縫的鍵盤熱鍵偵測（直接按 `A`, `D`, `S` 等即可操作）；若於 macOS 或 Linux 執行，為保護系統不崩潰，會自動切換為終端機 `Enter` 輸入模式。
* **防爆機制與效能**：
系統預設每處理 50 筆網址就會自動重啟一次背景 Chrome 瀏覽器，以釋放記憶體並防止程式崩潰（OOM），此為正常現象，請耐心等候。
* **JavaScript 動態標註**：
系統內建了強大的 DOM 穿透與 JavaScript 標註技術（自動展開手風琴選單、強制顯示隱藏元素）。但若目標網站採用高度封裝的 Canvas 或嚴格的防爬蟲阻擋機制（如 Cloudflare 驗證碼），可能會導致標註失敗或連線逾時，此時建議利用「模式 3」進行人工覆核。

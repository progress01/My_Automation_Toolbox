# 📄 Excel 報表合併與嚴格去重工具 (README)

本工具為基於 Python 與 `pandas` 開發的資料處理腳本。主要功能為批次讀取資料夾內的多份 Excel 報表，將其上下垂直合併後，針對「特定欄位組合」進行嚴格的重複值剔除（Deduplication），並自動重新編排流水號，最後匯出為一份乾淨的總表。

## 1. 需準備的輸入資料 (Input Data)

本程式採取資料夾批次讀取模式，請依循以下規範準備您的檔案：

* **資料夾設定**：請在程式執行的同一個目錄下，建立一個名為 **`excel_folder`** 的資料夾，並將所有需要合併的 Excel 檔案（支援 `.xlsx` 或 `.xls`）放入其中。
* **報表格式嚴格要求**：
1. **表頭略過機制**：程式內建 `skiprows=3` 設定。這意味著每份 Excel 的**第 1 到 3 行將被視為無用的報告標頭資訊而直接丟棄**。真正的資料標題列（Header）必須固定在**第 4 行**。
2. **必備欄位**：報表的標題列中，必須確切包含 **`連結網址`**、**`首次發現來源`** 以及 **`編號`** 這三個欄位名稱，否則在執行去重或流水號重編時會發生錯誤。



## 2. 預期的產出結果 (Expected Output)

執行完畢後，程式會在同一個目錄下自動生成一份名為 **`合併去重結果.xlsx`** 的 Excel 總表。

* **資料淨化**：所有被判定為重複的資料列皆已被移除，僅保留第一筆出現的紀錄。
* **流水號重置**：原先分散在各報表中的 `編號` 欄位將被全部清除，並依據合併後的資料總數，從 `1` 開始重新進行連續編號。
* **終端機戰報**：執行過程中，命令提示字元會即時印出「成功讀取了幾個檔案」、「合併前總筆數」、「去重後的最終筆數」以及「共移除了多少筆重複資料」等統計資訊。

## 4. 執行環境與注意事項 (Caveats & Requirements)

### 環境依賴 (Dependencies)

本腳本依賴強大的資料分析套件 `pandas` 以及 Excel 讀寫引擎。請於終端機執行以下指令進行安裝：

```bash
pip install pandas openpyxl

```

*(註：本腳本也支援讀取舊版的 `.xls` 檔案，若執行時報錯提示缺少 xlrd，可額外執行 `pip install xlrd`)*

### ⚠️ 執行注意事項

* **資料夾必須存在**：腳本並未內建自動建立 `excel_folder` 資料夾的語法。如果該資料夾不存在，或裡面沒有任何 Excel 檔案，程式會直接印出警告並停止執行。
* **嚴格的去重邏輯 (AND 條件)**：程式的去重條件為 `['連結網址', '首次發現來源']`。這表示**只有當「這兩個欄位的值同時一模一樣」時，才會被系統判定為重複**。若相同的連結出現在不同的發現來源，系統會將其視為兩筆不同的獨立事件並雙雙保留。
* **資料覆蓋風險**：每次執行腳本時，若目錄下已存在名為 `合併去重結果.xlsx` 的檔案，系統將會**直接覆蓋**舊檔案而不會另外警告。建議在下次執行前，先將舊的合併結果改名或移出資料夾。

# 📄 Word 查核報告查核審查閱卷版本

本工具為基於 Python 開發的高階查核轉換器。主要功能為批次讀取資料夾內的 Word 查核報告（`.docx` / `.doc`），自動在背景將其轉換為 PDF 以供視覺預覽，並透過 `mammoth` 與 `BeautifulSoup` 深度解析文件內容（包含萃取所有的外部連結與計分表格）。最終，系統會自動編譯出一個支援全鍵盤操作、自帶巨集與 LocalStorage 存檔機制的「沉浸式 HTML 單頁應用 (SPA) 儀表板」，極大地提升多檔審查的作業效率。

## 1. 需準備的輸入資料 (Input Data)

本程式採用資料夾批次讀取模式，請依循以下規範準備您的環境與檔案：

* **系統環境要求 (極重要)**：由於本程式使用了 `win32com.client` 來進行高品質的 PDF 轉換，**必須在安裝有 Microsoft Word 的 Windows 作業系統上執行**。
* **資料夾設定**：請將所有需要審查的 Word 報告（`.docx` 或 `.doc`）集中放置於同一個資料夾中。程式執行時，會彈出視窗請您選擇該資料夾。
* **文件格式要求**：
1. **基本中繼資料**：程式會自動掃描 Word 段落中帶有「網站名稱/網址」、「巡檢日期」、「網站資料」等關鍵字的文字作為報表標題。
2. **表格與項次**：程式會掃描文件中的表格，並尋找開頭帶有數字編號（例如 `1.` 或 `1、`）的儲存格。系統會將這些數字與內建的 22 項 `HARDCODED_RULES` (如：基本要件、導覽功能等) 進行映射匹配，自動帶入對應的評分標準與說明。



## 2. 預期的產出結果 (Expected Output)

執行程式並選擇資料夾後，系統會自動進行轉檔與解析，最後在該資料夾內生成以下產出物：

1. **📄 對應的 PDF 預覽檔 (`.pdf`)**
* 系統會自動檢查資料夾，若發現 Word 檔缺少同名的 PDF 檔，會在背景自動驅動 Microsoft Word 進行靜默轉檔，以供後續儀表板左側預覽使用。


2. **網頁查核計分板 (`.html`)**
* 根據您在啟動時選擇的模式，系統會生成 **`個人提報缺失表計分板.html`** (提報查驗模式) 或 **`單位成績單與申覆總表計分板.html`** (申覆審查模式)。
* **儀表板功能**：左側為 PDF 實體文件預覽（並自動將 Word 內附的超連結提取至畫面上方供獨立點擊）；右側為互動式計分板，支援全鍵盤操作、自訂備註巨集與即時進度存檔。


3. **最終審查總表 (`.csv`)**
* 在 HTML 儀表板中完成查核後，點擊「一鍵匯出 CSV 總表」，瀏覽器將自動下載一份包含所有檔案審查結果與備註的 CSV 彙整表。



## 3. 鍵盤快捷鍵與巨集操作 (Hotkeys & Macros)

儀表板專為「無滑鼠、高效率」的工作流打造。請確保**鍵盤焦點處於右側計分板**（右上角狀態燈號顯示「🟢 鍵盤就緒」）以使用以下快捷鍵：

| 快捷鍵 | 對應動作 |
| --- | --- |
| **1** | 標記為：✅ 過關 / 接受申覆，並自動跳至下一題。 |
| **2** | 標記為：❌ 缺失 / 駁回申覆，並自動跳至下一題。 |
| **3** | 標記為：➖ 略過 (無須審查)，並自動跳至下一題。 |
| **W / S** | 往上 / 往下切換題目焦點。 |
| **A / D** | 往左 / 往右切換下一個「Word 報告檔案」。 |
| **F2** | 快速將輸入游標定位至當前題目的「備註輸入框 (Textarea)」。 |
| **/** | (斜線) 快速聚焦至頂部的「檔案搜尋列」，可輸入關鍵字篩選檔案。 |
| **英文字母 (自訂)** | 觸發設定好的巨集 (Macro) 字串，瞬間填入備註並可設定是否連招（自動判定缺失 + 跳下一題）。 |

## 4. 執行環境與安裝指令 (Dependencies)

請確保您的 Python 環境已安裝以下外部套件。請於命令提示字元執行：

```bash
pip install mammoth beautifulsoup4 pywin32

```

*(註：`pywin32` 是讓 Python 能夠控制 Windows COM 介面與 Microsoft Word 的必備套件)*

# 📄 環境部新聞自動化爬蟲工具

本工具為基於 Python 與 Selenium 開發的互動式網頁爬蟲腳本。專門針對環境部（或具備相似 DOM 結構之政府網站）的新聞列表進行自動化翻頁與資料擷取。具備防呆機制（如頁面重複偵測、動態網址分頁參數修復），並能自動將民國年份轉換為西元年份，最終輸出乾淨、去重且依日期排序的 CSV 資料集。

## 1. 需準備的輸入資料 (Input Data)

本腳本採用**互動式命令列 (CLI) 提示**，不需要預先準備外部的設定檔或清單。請在終端機執行程式後，依序貼上或輸入以下四項資訊：

1. **開始日期**：純數字格式，例如 `20200101`（程式會自動轉為 2020/01/01）。
2. **結束日期**：純數字格式，例如 `20241231`。*(註：若起迄日期大小寫反，程式具備自動對調的防呆機制)*。
3. **要爬取的頁數**：純數字，例如 `5`（代表系統最多會往後翻 5 頁）。
4. **目標網址**：請直接貼上您在瀏覽器上的網址。
* *格式支援*：無論網址是否包含分頁參數（如 `page=1`、`PAGE=2`），或完全沒有分頁參數，腳本內建的正則表達式 (Regex) 皆能自動捕捉並動態替換為正確的翻頁模板。



## 2. 預期的產出結果 (Expected Output)

腳本執行完畢後，會在程式執行的同一個資料夾目錄下，自動生成一份名為 **`environment_news_manual_fixed.csv`** 的資料集（採 UTF-8-SIG 編碼，可直接用 Excel 開啟不會亂碼）。

**資料集包含以下 5 個欄位，並已自動剔除重複資料且依日期由新到舊排序：**

| 欄位名稱 | 說明 |
| --- | --- |
| **Publish_Date** | 發布日期。已將網頁上的民國年（如 112-01-01）自動轉換為西元年格式（如 2023/01/01）。 |
| **News_Title** | 新聞標題。*(系統會自動略過標題含有「更新日期」的無效列)* |
| **Category** | 消息分類（例如：最新消息、活動公告等）。 |
| **View_Count** | 點閱次數。已過濾掉千分位逗號（如 1,234 轉為 1234）。若網頁無提供則預設為 0。 |
| **Department** | 發布單位（例如：大氣環境司）。已清洗掉原始碼中多餘的括號 `[]` 並替換為頓號。 |

## 3. 執行環境與注意事項 (Caveats & Requirements)

### 環境依賴 (Dependencies)

請確保您的 Python 環境已安裝以下套件。請於命令提示字元執行：

```bash
pip install selenium webdriver-manager

```

*(註：`time`, `re`, `csv` 皆為 Python 內建標準庫，無需額外安裝)*

### 注意事項

* **瀏覽器綁定**：本程式依賴 Google Chrome 瀏覽器。執行時會自動彈出受控的 Chrome 視窗，請勿手動關閉或干擾該視窗，以免爬蟲中斷。`webdriver_manager` 會自動幫您處理驅動程式的版本對應問題。
* **前端結構依賴 (DOM 耦合)**：本腳本的資料解析高度依賴目標網站的 CSS 選擇器（例如：`ul.average-list > li`, `.date`, `.view-wrap`, `.gov-group`, `.classify`）。若未來該政府網站進行大幅度的前端改版，腳本中的 `find_element` 區塊將需要同步維護。
* **防無限迴圈機制**：部分網站當頁數超過總頁數時，會自動停留在最後一頁或導回第一頁。本腳本內建了「首筆標題記憶比對」，若偵測到當前頁面的第一筆新聞標題與上一頁完全相同，會觸發煞車機制並提早結束爬取，保障系統安全。
